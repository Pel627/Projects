# Python Network Scanner

Simple network scanner that detects active devices on a local network.

## Features

- Fast scanning using multithreading
- Detects active IP addresses
- Lightweight and simple

## Usage

1. Modify the network range in the script:

   network = "192.168.1."

2. Run the script:

   python network_scanner.py

## Example Output

[+] 192.168.1.1 está activa
[+] 192.168.1.5 está activa
